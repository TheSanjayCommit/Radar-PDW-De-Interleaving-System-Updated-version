OFFLINE INSTALLATION INSTRUCTIONS
=================================

1. Copy this entire 'offline_packages' folder to your offline PC (inside your project directory).

2. Open a terminal (Command Prompt or PowerShell).

3. Navigate to your project directory.

4. Run the following command to install the FPDF library from this folder:

   pip install offline_packages/fpdf-1.7.2.tar.gz

5. Once installed, start the application:

   streamlit run app.py

---------------------------------
Note: If the filename version changes, just type "pip install offline_packages/f" and press Tab to auto-complete.
